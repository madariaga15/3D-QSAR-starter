# 3D-QSAR starter

Starter 3D-QSAR workflow: RDKit 3D descriptors, alignment, and a random forest baseline.

## Run in Colab

Open the notebook:
https://colab.research.google.com/github/madariaga15/3D-QSAR-starter/blob/main/3d_qsar_starter.ipynb

## Repo contents

- `3d_qsar_starter.ipynb` — runnable Colab notebook
- `data/sample_dataset.csv` — toy SMILES + `target` column
- `requirements.txt` — Python dependencies

## Getting started (local)

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
